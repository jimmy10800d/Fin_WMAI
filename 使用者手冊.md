## 1. 系統簡介

### 什麼是 Fin_WMAI？

Fin_WMAI 是一套以**人生目標為導向**的智慧投資理財規劃系統。不同於傳統的投資平台，我們從您的人生目標出發，而非從複雜的金融商品開始。

### 核心特色

| 特色 | 說明 |
|------|------|
| 🎯 目標導向 | 從退休、買房、教育金等生活目標開始規劃 |
| 🗣️ 白話說明 | AI 用您聽得懂的語言解釋投資建議 |
| ⚠️ 風險優先 | 先揭露風險，讓您充分了解後再決定 |
| 📊 視覺化 | 清晰的圖表追蹤您的理財進度 |

## 2. 登入系統

### 2.1 進入登入頁面

開啟瀏覽器，前往系統登入頁面。

### 2.2 登入流程

**步驟一：輸入帳號**
1. 在「帳號 ID」欄位輸入您的使用者帳號
2. 點擊「下一步」按鈕

**步驟二：QR Code 身份驗證**
1. 系統會顯示一個 QR Code
2. 使用您的手機 App 掃描 QR Code
3. 在手機上確認登入

**步驟三：完成登入**
- 驗證成功後，系統會自動跳轉至首頁

### 2.3 Demo 帳號

如果您是首次體驗，可以使用以下帳號：

| 帳號 | 說明 |
|------|------|
| `user001` | 一般使用者（官大大） |
| `user002` | 一般使用者（陳小曦雲） |
| `demo` | 展示帳號 |

## 4. 目標設定

### 4.1 選擇人生目標

系統提供 8 種常見的人生理財場景：

| 場景 | 說明 |
|------|------|
| 🏠 買房頭期款 | 為購屋儲蓄首付款 |
| 🎓 子女教育金 | 規劃孩子的教育基金 |
| 🌴 退休生活 | 準備退休後的生活費用 |
| ✈️ 夢想旅遊 | 實現旅遊夢想的資金 |
| 🚗 購車基金 | 儲蓄購買新車的費用 |
| 💍 結婚基金 | 準備婚禮與新生活費用 |
| 🏥 醫療備用金 | 建立健康保障資金 |
| 🎨 自訂目標 | 設定您的專屬目標 |

### 4.2 設定目標參數

選擇目標後，需要填寫：

1. **目標金額**：您希望達成的金額（例如：200 萬）
2. **預計期程**：計劃達成的時間（例如：5 年）
3. **每月投入**：每月可投入的金額（例如：2 萬）

### 4.3 目標試算

系統會根據您的輸入進行試算，顯示：
- 預期達成率
- 建議投資方式
- 風險等級建議

## 5. 風險評估

### 5.1 KYC 問卷

風險評估包含 5 題問卷，用於了解您的：
- 投資經驗
- 風險承受度
- 投資目的
- 資金來源
- 投資期限

### 5.2 作答方式

1. 閱讀每題問題
2. 選擇最符合您情況的選項
3. 點擊「下一題」繼續
4. 完成所有題目後提交

### 5.3 評估結果

完成問卷後，系統會顯示：

| 項目 | 說明 |
|------|------|
| 風險分數 | 0-100 分的量化評估 |
| 風險等級 | 保守型/穩健型/積極型/進取型 |
| Gap 分析 | 目標與現況的差距分析 |
| 調整建議 | 達成目標的可行方案 |

### 5.4 調整方案

如果您的風險等級與目標有差距，系統會建議：
- 📅 **延長期程**：增加投資時間
- 💰 **增加投入**：提高每月投入金額
- 👤 **諮詢專人**：預約理財專員協助

## 6. AI 投資建議

### 6.1 建議內容

AI 會生成包含以下內容的投資建議書：

| 欄位 | 說明 |
|------|------|
| 配置理由 | 為何此方案適合您 |
| 投資組合 | 建議的資產配置比例 |
| 風險情境 | 用生活語言描述可能的波動 |
| 最壞狀況 | 極端情況下的可能損失 |
| 注意事項 | 重要的投資提醒 |

### 6.2 風險揭露

在查看具體建議前，您必須先：
1. 閱讀風險揭露聲明
2. 勾選「我已理解」
3. 才能查看詳細建議

### 6.3 聽不懂？換個方式說

如果您覺得建議內容難以理解，可以：
1. 點擊「🔄 換個方式說」按鈕
2. AI 會用不同的比喻重新解釋
3. 例如：運動比喻、園藝比喻、導航比喻

### 6.4 信任溫度計

您可以回饋對建議的信任程度：
- 😊 非常信任
- 🙂 還算信任
- 😐 普通
- 😕 有些疑慮
- 😟 不太信任

## 7. 交易執行

### 7.1 查看行動清單

根據 AI 建議，系統會生成可執行的行動清單：

```
┌─────────────────────────────────────┐
│ 📋 投資行動清單                      │
├─────────────────────────────────────┤
│ ✅ 開立基金帳戶                      │
│ ⏳ 設定每月定期定額 NT$20,000        │
│ ○ 配置全球股票型基金 60%             │
│ ○ 配置債券型基金 40%                 │
└─────────────────────────────────────┘
```

### 7.2 風控檢核

執行交易前，系統會進行：
- **適配性檢核**：確認商品符合您的風險等級
- **額度檢核**：確認可用資金充足
- **合規檢核**：確保符合法規要求

### 7.3 確認執行

1. 點擊「執行交易」按鈕
2. 確認交易明細
3. 輸入交易密碼（如需要）
4. 完成交易

## 11. AI 助理管理

點擊側邊欄「AI 助理管理」（🧠）即可進入管理頁面，包含三個分頁：

### 11.1 對話記憶

- **查看**：瀏覽您與小曦雲的所有歷史對話
- **釘選**：釘選重要對話以便日後查閱
- **刪除**：移除單筆記錄或清除全部

### 11.2 排程任務

設定定期提醒，如：
- 📊 **每週日戰績回報**：每週日早上 9:00 自動產生投資戰績摘要
- 🎯 **里程碑進度檢查**：定期檢查目標達成狀況
- 🌡️ **投資心情回報**：定期關懷您的投資心理狀態

您可以：
1. **新增排程**：選擇頻率與類型
2. **立即執行**：手動觸發產生即時報告
3. **暫停/啟用**：控制排程是否執行

### 11.3 計畫項目

查看您的所有投資目標與里程碑：
- **目標追蹤**：顯示進度條、目前金額、每月目標
- **里程碑**：查看已達成與未達成的里程碑
- **新增目標**：自定義您的理財目標

## 13. 常見問題

### Q1：忘記密碼怎麼辦？

請點擊登入頁面的「忘記密碼」連結，或聯繫客服。

### Q2：QR Code 掃描失敗？

1. 確認手機 App 已更新至最新版本
2. 點擊「重新產生」獲取新的 QR Code
3. 確保手機相機對準 QR Code

### Q3：AI 建議一定要接受嗎？

不是的。AI 建議僅供參考，您可以：
- 調整建議內容
- 諮詢理財專員
- 自行決定投資方式

### Q4：如何修改個人資料？

1. 點擊側邊欄底部的用戶資訊
2. 選擇「個人設定」
3. 修改所需資料
4. 儲存變更

### Q5：交易可以取消嗎？

- **處理中**：可在一定時間內取消
- **已成交**：無法取消，需另行處理

### Q6：系統安全嗎？

系統採用多重安全機制：
- 🔐 加密傳輸
- 🛡️ 雙因素認證
- 📝 完整稽核軌跡
- 🔒 個資保護
